HTML5 Features with browser support

HTML5 - Canvas

The HTML5 <canvas> element is used to draw graphics, on the fly, via scripting (usually JavaScript). The <canvas> element is only a container for graphics. You must use a script to actually draw the graphics. Canvas has several methods for drawing paths, boxes, circles, text, and adding images.

HTML5 - SVG

         
SVG stands for Scalable Vector Graphics. It is used to define vector-based graphics for the Web. It defines the graphics in XML format. It's graphics do NOT lose any quality if they are zoomed or resized. SVG images can be created and edited with any text editor
Example
 
 

HTML5 - Drag and Drop

Drag and drop is a very common feature. It is when you "grab" an object and drag it to a different location. In HTML5, drag and drop is part of the standard, and any element can be draggable.
